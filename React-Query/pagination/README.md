# React Query(Tanstack Query) - 페이지네이션

### 사용 기술 및 API

- 사용 기술 : React, TypeScript, React Query(Tanstack Query)
- 사용 API : <a href='https://openlibrary.org/dev/docs/api/search'>Open Library Search API</a>

### 목표

- React Query를 사용하여 페이지네이션을 구현합니다.
- useQuery를 활용합니다.

### 설치 및 실행

```
$ npm install
$ npm run start
```
